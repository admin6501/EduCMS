## 🚀 نصب سریع

### پیش‌نیازها

قبل از نصب، مطمئن شوید که:

- ✅ سیستم‌عامل: **Ubuntu 20.04 / 22.04 / 24.04**
- ✅ دسترسی: **root** یا **sudo**
- ✅ دامنه: متصل به IP سرور
- ✅ پورت‌ها: **80** و **443** باز باشند

> 💡 **نکته:** تمام وابستگی‌ها (Docker، Docker Compose، SSL و ...) به‌صورت **خودکار** نصب می‌شوند.

### نصب با یک دستور

برای نصب EduCMS کافیست دستور زیر را در ترمینال سرور اجرا کنید:

```bash
bash <(curl -sSL https://raw.githubusercontent.com/admin6501/EduCMS/refs/heads/main/install.sh)
```

پس از اجرا، منوی تعاملی نمایش داده می‌شود و اسکریپت شما را **مرحله‌به‌مرحله** راهنمایی می‌کند.


## 📚 راهنمای استفاده

### 🔐 پنل ادمین

پس از نصب، به پنل ادمین دسترسی پیدا کنید:

```
https://yourdomain.com/admin/
```

> ⚠️ **امنیت:** توصیه می‌شود مسیر پیش‌فرض `/admin/` را تغییر دهید.

**امکانات پنل ادمین:**
- مدیریت دوره‌ها و دسته‌بندی‌ها
- تایید و رد پرداخت‌ها
- مدیریت سفارش‌ها و کاربران
- مدیریت تیکت‌ها
- ویرایش تنظیمات سایت

### 💾 پشتیبان‌گیری از دیتابیس

**گرفتن بکاپ:**

1. اسکریپت را اجرا کنید
2. گزینه `5) Backup DB` را انتخاب کنید
3. فایل بکاپ در مسیر زیر ذخیره می‌شود:

```
/opt/educms/backups/educms-YYYYMMDD-HHMMSS.sql
```

### ♻️ بازگردانی دیتابیس

**ریستور از بکاپ:**

1. اسکریپت را اجرا کنید
2. گزینه `6) Restore DB` را انتخاب کنید
3. مسیر کامل فایل بکاپ را وارد کنید

> ⚠️ **توجه:** ریستور باعث **حذف کامل** دیتابیس فعلی می‌شود.
