[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

**EduCMS** یک سیستم مدیریت آموزش (LMS) کامل، آماده‌ی استفاده در محیط واقعی است که با **Django** و **Docker** توسعه داده شده و به‌صورت پیش‌فرض برای کاربران **فارسی‌زبان** طراحی شده است.

## 📖 درباره پروژه

این پروژه با هدف **نصب سریع**، **مدیریت آسان** و **امکانات کامل** ساخته شده است. تمام مراحل راه‌اندازی شامل SSL، دیتابیس و سرویس‌ها تنها با **یک دستور** انجام می‌شود.

### چرا EduCMS؟

- ✅ **نصب در کمتر از ۵ دقیقه** - بدون نیاز به دانش فنی پیچیده
- ✅ **کاملاً فارسی و RTL** - طراحی شده برای کاربران ایرانی
- ✅ **آماده Production** - با Nginx، Gunicorn و SSL خودکار
- ✅ **مبتنی بر Docker** - قابل حمل و یکپارچه

## 🚀 نصب سریع

### پیش‌نیازها

قبل از نصب، مطمئن شوید که:

- ✅ سیستم‌عامل: **Ubuntu 20.04 / 22.04 / 24.04**
- ✅ دسترسی: **root** یا **sudo**
- ✅ دامنه: متصل به IP سرور
- ✅ پورت‌ها: **80** و **443** باز باشند

> 💡 **نکته:** تمام وابستگی‌ها (Docker، Docker Compose، SSL و ...) به‌صورت **خودکار** نصب می‌شوند.

### نصب با یک دستور

برای نصب EduCMS کافیست دستور زیر را در ترمینال سرور اجرا کنید:

```bash
bash <(curl -sSL https://raw.githubusercontent.com/admin6501/EduCMS/refs/heads/main/install.sh)
```

پس از اجرا، منوی تعاملی نمایش داده می‌شود و اسکریپت شما را **مرحله‌به‌مرحله** راهنمایی می‌کند.

---

## 🧭 منوی مدیریت

پس از اجرای اسکریپت، منوی زیر نمایش داده می‌شود:

```
============================================
            EduCMS Menu (Latest)            
============================================
Path: /opt/educms

1) Install        (نصب کامل)
2) Patch          (بروزرسانی کد)
3) Start          (استارت)
4) Stop           (توقف)
5) Restart        (ری‌استارت)
6) Backup DB      (.sql)
7) Restore DB     (.sql)
8) Uninstall      (حذف کامل)
0) Exit

Select: 
```

## 📚 راهنمای استفاده

### 🚀 مراحل نصب

هنگام اجرای گزینه `1) Install`، اسکریپت اطلاعات زیر را از شما می‌پرسد:

| مرحله | توضیحات | مثال |
|-------|---------|------|
| **دامنه** | آدرس دامنه سایت شما | `example.com` |
| **ایمیل** | برای دریافت SSL از Let's Encrypt | `admin@example.com` |
| **مسیر ادمین** | آدرس پنل مدیریت (پیش‌فرض: admin) | `mypanel` |
| **نام دیتابیس** | نام پایگاه داده (پیش‌فرض: educms) | `educms` |
| **کاربر دیتابیس** | نام کاربری دیتابیس | `dbuser` |
| **رمز دیتابیس** | رمز عبور دیتابیس | `******` |
| **نام کاربری ادمین** | برای ورود به پنل مدیریت | `admin` |
| **رمز ادمین** | رمز عبور پنل مدیریت | `******` |

> 💡 پس از وارد کردن اطلاعات، نصب به‌صورت خودکار انجام می‌شود.

### 🔐 پنل ادمین

پس از نصب، به پنل ادمین دسترسی پیدا کنید:

```
https://yourdomain.com/admin/
```

یا اگر مسیر را تغییر داده‌اید:

```
https://yourdomain.com/your-custom-path/
```

> ⚠️ **امنیت:** توصیه می‌شود مسیر پیش‌فرض `/admin/` را تغییر دهید.

#### امکانات پنل ادمین:

| بخش | امکانات |
|-----|---------|
| **دوره‌ها** | ایجاد، ویرایش، حذف دوره‌ها و مدیریت جلسات |
| **دسته‌بندی‌ها** | سازماندهی دوره‌ها در گروه‌های مختلف |
| **کاربران** | مشاهده، ویرایش و مدیریت کاربران |
| **سفارش‌ها** | مشاهده سفارشات و تایید/رد پرداخت‌ها |
| **تیکت‌ها** | پاسخ‌دهی به درخواست‌های کاربران |
| **تنظیمات** | ویرایش متن‌ها، فوتر، لوگو و لینک‌ها |
| **کدهای تخفیف** | ایجاد و مدیریت کدهای تخفیف |

### 📚 مدیریت دوره‌ها

#### ایجاد دوره جدید:
1. وارد پنل ادمین شوید
2. به بخش **دوره‌ها** بروید
3. روی **افزودن دوره** کلیک کنید
4. اطلاعات دوره را وارد کنید:
   - عنوان دوره
   - توضیحات
   - قیمت
   - تصویر کاور
   - دسته‌بندی
5. دوره را ذخیره کنید

#### افزودن جلسه به دوره:
1. دوره مورد نظر را انتخاب کنید
2. به بخش **جلسات** بروید
3. روی **افزودن جلسه** کلیک کنید
4. ویدیو را آپلود کنید یا لینک خارجی وارد کنید

### 💳 سیستم پرداخت

EduCMS از سیستم پرداخت **کارت‌به‌کارت** پشتیبانی می‌کند:

1. کاربر دوره را انتخاب و خرید می‌کند
2. اطلاعات کارت مقصد نمایش داده می‌شود
3. کاربر مبلغ را واریز و رسید را آپلود می‌کند
4. ادمین پرداخت را بررسی و تایید/رد می‌کند
5. پس از تایید، دسترسی کاربر فعال می‌شود

#### تنظیم اطلاعات پرداخت:
از پنل ادمین > تنظیمات > اطلاعات پرداخت

### 🎟️ کدهای تخفیف

#### انواع تخفیف:

|-----|---------|

#### ایجاد کد تخفیف:
1. پنل ادمین > کدهای تخفیف
2. افزودن کد جدید
3. تنظیم نوع، مقدار و تاریخ انقضا

### 🎫 سیستم تیکتینگ

کاربران می‌توانند از طریق سیستم تیکت با پشتیبانی ارتباط برقرار کنند:

**برای کاربران:**
- ثبت تیکت جدید با موضوع و توضیحات
- امکان پیوست فایل
- مشاهده تاریخچه تیکت‌ها

**برای ادمین:**
- مشاهده تمام تیکت‌ها
- پاسخ‌دهی به تیکت‌ها
- تغییر وضعیت (باز/بسته)

---

### 🔄 بروزرسانی سیستم (Patch)

برای بروزرسانی کد بدون از دست دادن داده‌ها:

1. اسکریپت را اجرا کنید
2. گزینه `2) Patch` را انتخاب کنید
3. سیستم به‌صورت خودکار بروزرسانی می‌شود

> 💡 این گزینه فقط کد را بروزرسانی می‌کند و داده‌های شما حفظ می‌شود.

---

### ▶️ استارت / توقف / ری‌استارت

| گزینه | کاربرد |
|-------|--------|
| `3) Start` | راه‌اندازی سرویس‌ها (پس از Stop) |
| `4) Stop` | متوقف کردن تمام سرویس‌ها |
| `5) Restart` | ری‌استارت کردن سرویس‌ها |

---

### 💾 پشتیبان‌گیری از دیتابیس

**گرفتن بکاپ:**

1. اسکریپت را اجرا کنید
2. گزینه `6) Backup DB` را انتخاب کنید
3. فایل بکاپ در مسیر زیر ذخیره می‌شود:

```
/opt/educms/backups/educms-YYYYMMDD-HHMMSS.sql
```

> 💡 توصیه می‌شود به‌صورت منظم از دیتابیس بکاپ بگیرید.

---

### ♻️ بازگردانی دیتابیس

**ریستور از بکاپ:**

1. اسکریپت را اجرا کنید
2. گزینه `7) Restore DB` را انتخاب کنید
3. مسیر کامل فایل بکاپ را وارد کنید:

```
/opt/educms/backups/educms-20250101-120000.sql
```

> ⚠️ **توجه:** ریستور باعث **حذف کامل** دیتابیس فعلی می‌شود.

---

### 🗑️ حذف کامل

برای حذف کامل EduCMS از سرور:

1. اسکریپت را اجرا کنید
2. گزینه `8) Uninstall` را انتخاب کنید
3. تایید کنید

> ⚠️ **هشدار:** این عملیات تمام داده‌ها، فایل‌ها و تنظیمات را حذف می‌کند و قابل بازگشت نیست!

### 🎨 شخصی‌سازی ظاهر
