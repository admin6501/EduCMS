🎓 EduCMS

EduCMS یک سیستم مدیریت آموزش (LMS) کامل و آماده‌ی استفاده در محیط واقعی است که با Django و Docker توسعه داده شده و به‌صورت پیش‌فرض برای کاربران فارسی‌زبان طراحی شده است.

این پروژه با هدف نصب سریع، مدیریت آسان و امکانات کامل ساخته شده و تمام مراحل راه‌اندازی، SSL، دیتابیس و سرویس‌ها را تنها با یک اسکریپت Bash انجام می‌دهد.

🚀 نصب با یک دستور (One-Line Install)

برای نصب EduCMS فقط دستور زیر را در ترمینال سرور اجرا کنید:

bash <(curl -sSL https://raw.githubusercontent.com/admin6501/EduCMS/refs/heads/main/install.sh)

بعد از اجرا، منوی تعاملی EduCMS نمایش داده می‌شود و اسکریپت شما را مرحله‌به‌مرحله راهنمایی می‌کند.


---

🧭 منوی اسکریپت نصب

پس از اجرای اسکریپت، منوی زیر نمایش داده می‌شود:

1) Install (نصب کامل)
2) Stop (توقف)
3) Restart (ری‌استارت)
4) Uninstall (حذف کامل)
5) Backup DB (.sql)
6) Restore DB (.sql)
0) Exit
